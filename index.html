Quickstart

Up and running in minutes

$ pip install autochecklist
from autochecklist import DirectGenerator, ChecklistScorer # Generate a checklist for your evaluation task generator = DirectGenerator() checklist = generator.generate( task="Evaluate the quality of this summary", input_text="The original article about climate change...", ) # Score a response against the checklist scorer = ChecklistScorer() result = scorer.score(checklist=checklist, response="The summary text...")
